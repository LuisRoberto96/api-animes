Projeto feito em React que realiza a busca de informações (utilização do método GET para a API) do site "kitsu.io".

O projeto conta com a utilização das funções do useState e useEffect do React Hooks,
bem como a utilização do Debounce para tratamento de requisições. 

Cada pesquisa apresenta 12 itens na interface do projeto (foi estabelecido um limite de 12 itens por pesquisa).
